Overview

This project is a microservices-based quiz management system built using Spring Boot. The application separates core functionality into independent services to ensure scalability and modularity.

Architecture

User Service

Quiz Service

Question Service

REST-based inter-service communication

Each service maintains its own database for better data isolation.

Tech Stack

Java

Spring Boot

REST APIs

Spring Data JPA

MySQL / H2

Maven

Git

Features

User management

Quiz creation and management

Question management

Microservice communication via REST

Independent database per service

Scalable modular design

How to Run

Clone repository

Configure databases for each service

Run each service individually

Access APIs using Postman
